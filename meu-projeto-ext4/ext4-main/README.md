# ext4
sistema de arquivos ext4 para a disciplina de SO

Como rodar no VS Code:

Abra o terminal do VS Code (Ctrl + ' ou pelo menu superior) e execute apenas o comando abaixo:

 gcc main.c -o meu_shell 

Para rodar seu interpretador passando o arquivo de imagem:

 ./meu_shell images_ext4/myext4image1k.img
